QRPSI

Questionários de Riscos Psicossociais · Resumo executivo

Plataforma web que substitui o ciclo Google Forms + Excel por dados centralizados, validação no servidor e indicadores padronizados.

Proposta de valor

  • 39 perguntas / 5 temas em catálogo único — comparabilidade entre empresas e ao longo do tempo.
  • Link público por pesquisa (UUID); respondente sem login; nome opcional.
  • Consolidação automática no SQL Server — elimina somatórias frágeis em planilha.
  • Alertas ≥ 35% “Sim” em pergunta, tema e painel global.

Módulos principais

  • Admin: empresas, usuários, pesquisas (status RASCUNHO → ABERTA → ENCERRADA/CANCELADA), dashboard.
  • Público: formulário Sim/Não com validação total no servidor e transação na gravação.
  • Departamentos: autocomplete + cadastro automático com nome normalizado.
  • Relatórios: por pesquisa (pergunta, tema, departamento) + gráficos; exportação Excel (.xlsx) com aba Metadados, filtros, % formatado e destaque vermelho para indicadores ≥35%; dashboard institucional.

Forms + Excel × QRPSI

Critério Antes Com QRPSI
Consolidação Export/manual; erros em fórmulas Agregações no backend; regras únicas
Completude Depende do Forms Todas as perguntas obrigatórias validadas na API
Corte organizacional Planilha difícil de padronizar Departamentos com normalização e relatório
Stack — Node.js (Express), SQL Server, React (Mantine), API REST + JWT
Exportação Excel Manual / copiar-colar 3 relatórios XLSX (pergunta, tema, departamento) direto na tela da pesquisa
